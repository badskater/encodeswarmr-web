# encodeswarmr-web

Website for the [EncodeSwarmr](https://github.com/badskater/encodeswarmr) project — a distributed video encoding orchestration platform.

Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Output goes to `./dist/`.

## Structure

- `src/pages/index.astro` — Landing page (hero, features, architecture, quick start)
- `src/pages/docs/` — Documentation pages (overview, installation, configuration, agents, API, HDR, webhooks, HA)
- `src/layouts/` — Layout components (main site layout + docs layout with sidebar)
